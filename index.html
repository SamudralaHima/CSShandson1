What is CSS?

  1. CSS stands for Cascading style sheets.
  2. CSS describes how HTML element are to be displayed on screen,paper, or in other media.
  3. Css is not neccesaryCSS saves a lot of work.It can control the layout of multiple web pages all at once
  4. External stylesheets are stored in CSS files.

CSS can be added to HTML documents in 3 ways:

  • Inline - by using the style attribute inside HTML elements1
  • Internal - by using a style elements in the head section
  • External - by using a link elements to link to an external CSS file.
  • The most common way to add CSS. is to keep the style in external CSS file. However, in this tutorial we will use inline and internal style, because this is easier to demonstrate, and easier for you to try it yourself.

When comparing CSS class vs ID, the difference is that CSS class applies a style to multiple elements. ID, on the other hand,applies a style to one unique element.ID is also spcial in that you can use a special in that you can use a spcial URL to link directly to an element and it's used by JavaScript.
